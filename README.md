# Trading Bot

Automated trading bot application.

## Getting Started

This project includes:
- Main trading bot application (main.py)
- Dashboard (dashboard.py)
- Data processing and analysis modules

## Installation

```bash
pip install -r requirements.txt
```

## Running

To run the trading bot:
```bash
python main.py
```

To run the dashboard:
```bash
streamlit run dashboard.py
```
